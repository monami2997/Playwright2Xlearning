# Playwright2Xlearning

Playwright examples and practice tests written in TypeScript.

## Test Topics

- [08 Web Select, Frames, and Iframe](tests/08_Web_Select_Frames_Iframe)
	- Select controls and custom dropdowns
	- Advanced select examples
- [09 Frame and Iframe](tests/09_Frame_Iframe)
	- Iframe interaction
	- Frame-set examples
- [10 Keyboard, Hover, and Drag-and-Drop](tests/10_Keyboard_Hover_Drag_Drop)
	- Keyboard actions and hover interactions
	- Drag-and-drop and context-menu examples
- [11 JavaScript Alerts](tests/11_JS_Alerts)
	- JavaScript dialog handling

Run the tests with:

```bash
npx playwright test
```
